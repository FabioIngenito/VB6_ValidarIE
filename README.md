# VB6_ValidarIE

VALIDAR INSCRICAO ESTADUAL POR ESTADO

' ---------------
' Dois Exemplos:
' 250768135 - SC
' 76105790  - RJ
' ---------------

Abaixo, um código errado ... 
NÃO é possível encontrar um ESTADO digitando uma IE. 
Infelizmente se você colocar '250768135' servirá para Amazonas, Ceará e outros estados além de Sta.Catarina. 
No Brasil tudo é mal feito, subdimensionado e superfaturado.
